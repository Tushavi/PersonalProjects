Personal Python Project - Percolation Simulator

* Before running the simulation, change the config variables as per your desire

* To watch a simultion, run command ``python simulator.py``
  - Enter size n of n*n grid.
  - Watch the simulation and the results.
  - Press return in terminal to exit.
* To look at the statistics of n simulations, run command ``python stats.py``
  - Enter size of n*n grid.
  - Get the results and a histogram.
  - Exit the histogram.
